# Gustavo Saturnino - Dracula Terminal Portfolio

A modern, interactive portfolio designed as a high-fidelity "Terminal Command Center".

## Features

- **Dracula Theme**: Support for Dark and Light modes (inverted Dracula palette).
- **Multi-Language**: Support for English and Portuguese content.
- **Interactive Terminal**: Run Unix-like commands (`cat`, `ls`, `cd`, `clear`, `whoami`, `date`, `lang`, `theme`) to explore the technical path.
- **Full-Screen UI**: Optimized for a focused "Command Center" experience.
- **Pixelated Profile**: 8-bit style profile photo integration.
- **Technical Path**: Focused on Software Engineering and technical support experience.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Theme**: Dracula
